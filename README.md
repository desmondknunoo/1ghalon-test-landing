# 1Ghalon Landing Page

This is the official landing page for the 1Ghalon mobile application. The page is designed to provide users with information about the app's features, mission, and provide download links for both iOS and Android platforms.

## About 1Ghalon

**1Ghalon** is a mobile app designed to show fuel prices and directions to nearby fuel stations in Ghana. It helps users make informed decisions by providing up-to-date fuel prices, sourcing data from CediRates.com.

## Landing Page Features

-   **Responsive Design**: Fully responsive layout that works on all devices, from mobile phones to desktop computers.
-   **Smooth Scrolling**: Easy navigation with smooth scrolling to different sections of the page.
-   **Dynamic Elements**: Includes a preloader, a "scroll to top" button, and a mobile navigation overlay for a better user experience.
-   **Social Media Ready**: Optimized with Open Graph and Twitter Card meta tags for rich social media previews.
-   **Clear Sections**:
    -   **Home**: Hero section with a clear call-to-action.
    -   **Features**: Detailed look at the app's main functionalities.
    -   **Team**: Information about the team behind the app.
    -   **Info**: Key highlights of the app's data sources and privacy.
    -   **Download**: Direct links to download the app from the App Store and Google Play.

## App Features Highlighted

-   **Easy Directions**: Find the nearest and cheapest fuel stations with ease.
-   **NPA News**: Get real-time updates from the National Petroleum Authority (NPA).
-   **Real-time Data**: Fuel prices are sourced from CediRates.com for accuracy.
-   **Privacy Focused**: The app respects user privacy by collecting only non-personal data to improve performance.
-   **Beta Access**: Currently available for public testing, including a TestFlight link for iOS users.

## Technologies Used

-   **HTML5**: For the structure of the web page.
-   **CSS3**: For styling, including a custom grid system.
-   **JavaScript**: For interactivity and dynamic features.
-   **jQuery**: Used for simplifying DOM manipulation and event handling.
-   **Google Fonts**: Utilizes the "Hanken Grotesk" font for typography.

## Setup

To run this project locally, follow these steps:

1.  Clone the repository to your local machine:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd 1ghalon-test-landing
    ```
3.  Open the `index.html` file in your web browser to view the landing page.
